下载

Nginx下载地址:http://nginx.org/en/download.html

环境安装

执行命令

yum install gcc-c++
yum install -y pcre pcre-devel
yum install -y zlib zlib-devel
yum install -y openssl openssl-devel

安装

上传后解压

tar -zxvf nginx-1.20.1.tar.gz

编译安装

进入nginx目录，执行命令

./configure –prefix=/opt/nginx
make
make install


prefix指定目录

启动nginx

进入/opt/nginx/sbin (指定的安装目录)
通过以下命令启动

./nginx

在浏览器输入ip出现下面界面就安装成功了！

常用命令

都是在/opt/nginx/sbin目录下执行！

查看版本号

[root@iZbp12qqpsd4bzvroh2erhZ sbin]# ./nginx -v
nginx version: nginx/1.20.1

启动

强制关闭

平缓关闭

重新加载nginx
