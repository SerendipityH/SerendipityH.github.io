CentOS安装Redis
发表于2022-01-07|更新于2022-05-28|Linux
|阅读量:

下载

官网下载网页链接：https://download.redis.io/releases/
选择需要的版本并上传。
使用命令解压

tar -zxvf redis-5.0.5.tar.gz

环境安装

由于在安装过程中需要对源码进行编译，而编译依赖 gcc 环境。

yum install gcc-c++

编译

进入解压后的目录执行make命令

进入src目录执行make install

编辑配置文件

将redis.conf 文件复制到 /opt/redis目录下，防止文件被破坏

然后进入/opt/redis下编辑 redis.conf文件

daemonize

设置daemonize，将其设置为yes，Redis设置为以守护进程的方式进行启动

连接图像化工具

将 bind 127.0.0.1 注释掉，将protected-mode设置为no

增加密码(可选)

requirepass password(password为自己设置的密码)

启动

命令启动,然后就可以用可视化工具连接了
