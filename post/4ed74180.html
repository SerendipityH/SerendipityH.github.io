下载安装包

方法一:官网下载安装包

官网地址：https://downloads.mysql.com/archives/community/

方法二:使用wget命令下载

解压

创建mysql用户组和用户并修改权限

groupadd mysql
useradd -r -g mysql mysql

更改mysql目录下所有的目录及文件夹所属的用户组和用户，以及权限

在/opt/mysql目录下创建data目录

编译安装并初始化MySQL

进入mysql的bin目录执行命令

basedir 参数
该参数指定了安装 MySQL 的安装路径，填写全路径可以解决相对路径所造成的问题。
datadir 参数
该参数指定了 MySQL 的数据库文件放在什么路径下。数据库文件即我们常说的 MySQL data 文件。

记住初始化密码

启动mysql

这里报错缺少目录/var/lib/mysql
解决方法:
创建目录赋予权限

再次启动提示成功

添加软连接，并重启mysql服务

登录数据库修改密码

执行命令

Enter password:输入密码(复制上面的临时密码)

登录时报错
解决方案：

在my.cnf增加如下配置
[client]
port=3306
socket=/var/lib/mysql/mysql.sock

再次启动就成功了
