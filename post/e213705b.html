创建SpringBoot项目

Spring Initializer

在IDEA中左上角File->New->Project
Spring Initializer 可以快速生成 SpringBoot项目
默认的地址改成https://start.aliyun.com (国内用阿里的比较快)

Project Metadata

然后设置项目坐标及项目名称

坐标Group ID是项目组织唯一的标识符，实际对应项目中的package包。
坐标Artifact ID是项目的唯一的标识符，实际对应项目的project name名称，Artifact不可包含大写字母。

创建Vue项目

下载Node

Node下载地址:http://nodejs.cn/download/
Node以往的版本地址: https://nodejs.org/zh-cn/download/releases/
下载后就正常的安装就行了

输入命令即可查看 node -v

安装vue-cli

卸载老版本(可略过)

npm uninstall -g @vue/cli
或
yarn global remove @vue/cli

这个卸载时间比较长

安装阿里云镜像

npm install -g cnpm –registry=https://registry.npm.taobao.org

查看vue版本

npm view @vue/cli versions –json

安装指定版本

选择一个稳定的版本安装

cnpm install -g @vue/cli@3.12.0

安装完成后 可输入vue -V命令查看

创建项目

vue create demo

手动选择预设

也可以用

vue ui
打开页面进行创建
